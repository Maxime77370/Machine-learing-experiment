# Projet Machine Learning 🤖

## Description

Ce projet, axé sur l'exploration du fonctionnement des machines d'apprentissage (Machine Learning), a été créé dans le but de comprendre plus en profondeur le monde fascinant de l'IA. Actuellement à la version V1.1, le projet se concentre sur la **Compartimentation et l'Amélioration**.

### Objectif

L'objectif principal de ce projet est de plonger dans les mécanismes d'un réseau de neurones simple, capable d'apprentissage automatique.

## Versions

### V1.1 - Compartimentation et Amélioration 🚀

Dans cette version, plusieurs améliorations et modifications ont été apportées au projet. Voici un aperçu :

- **Correction de bugs :**
  - Ajustement du nombre de couches et de neurones qui était mal pris en compte dans certaines boucles.

- **Ajouts :**
  - **Affichage du réseau de neurones :**
    - Positionnement clair des neurones et des liaisons.
    - Attribution de couleurs pour différents poids sur les liaisons.
      - Utilisation d'une fonction pour normaliser les données entre x = [0,1] : x = (y-y_min)/(y_max-y_min).
    - Attribution de couleurs pour les valeurs actuelles sur les liaisons (bogue identifié).

- **À faire :**
  - Ajout de poids sur chaque neurone plutôt que seulement sur les connexions.
  - Compartimenter le code en plusieurs classes : Play, Game, ML.
  - Recherche de bugs supplémentaires.

### V1.0 - Reprise du Projet 🔄

Dans cette version initiale, le projet a été repris et des ajustements ont été effectués, notamment :

- Correction de bugs liés à la mauvaise prise en compte du nombre de couches et de neurones.
- Ajout d'un affichage du réseau de neurones avec des fonctionnalités spécifiques.
  - Positionnement des neurones et des liaisons.
  - Attribution de couleurs pour différents poids sur les liaisons (fonction de normalisation utilisée).
  - Attribution de couleurs pour les valeurs actuelles sur les liaisons (bogue identifié).

## À venir

Le projet continuera d'évoluer avec les futures versions, mettant l'accent sur l'ajout de fonctionnalités, la correction de bugs et l'optimisation globale du réseau de neurones.

N'hésitez pas à contribuer, signaler des bugs ou suggérer des améliorations ! 🙌
